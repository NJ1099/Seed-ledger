기기 간 동기화

이 인스턴스에는 텔레그램 봇이 설정되지 않았어. 운영자가 TELEGRAM_BOT_TOKEN 환경 변수를 추가하면 다른 PC·모바일 간 동기화가 가능해져.

텔레그램 계정을 한 번 연동해두면 모든 자산·거래·스냅샷이 너의 텔레그램 채팅에 암호화 백업돼. 다른 PC 에서 같은 텔레그램으로 페어링하면 그대로 복원.

  • 서버는 데이터를 갖고 있지 않음 — 봇이 너의 채팅에 JSON 파일을 보내고 핀(고정)함
  • 비밀번호 없음 — 텔레그램 계정 자체가 인증 수단
  • 해제는 언제든 가능, 텔레그램에서 봇 차단으로도 가능
